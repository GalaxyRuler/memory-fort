## IP ownership of AI-assisted work

Per the terms of service of the AI providers used:

- **Anthropic** ([Commercial Terms of Service](https://www.anthropic.com/legal/commercial-terms)): output generated by Claude on the user's behalf belongs to the user
- **OpenAI** ([Terms of Use](https://openai.com/policies/terms-of-use)): the user owns output generated by their use of the service, subject to compliance with the terms

Accordingly, all output of the AI coding agents that contributed to Memory Fort is owned by Abdullah Al Kulaib as the prompting user. No AI provider holds any ownership stake in Memory Fort.

## Human-authored portions

Direction, architecture decisions, design choices, code review, brief authoring, and the substantive decisions that shape Memory Fort were made by the project author. The AI agents executed implementation tasks within scopes defined by author-written briefs (see `docs/codex-*.md` for the implementation briefs and `docs/ROADMAP.md` for the strategic direction).

## Future contributors

Memory Fort is currently a sole-authored project. If outside contributions are accepted in the future, they will be accepted under the same GPLv3 terms that cover the rest of the project unless a separate written agreement says otherwise.

Until a CLA process is in place, pull requests from outside contributors will be discussed individually with the author before being merged.

Every commit in Memory Fort is signed by `GalaxyRuler <aoa@live.ca>` as author and committer. AI-assisted commits include the `Co-Authored-By:` trailer in the commit message body. The git history is the authoritative record of who contributed what and when.
