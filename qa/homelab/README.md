# Homelab QA

Project-owned Homelab runner profiles for container, CLI, API, unit, integration, and headless checks live here. Do not put project-specific profile details in Homelab core.

No live full-suite profile is currently configured. The full `npm test` suite
has Windows-specific path and tool assumptions, so a Linux container profile is
not equivalent until those tests are made portable or a Windows runner profile
is added.
