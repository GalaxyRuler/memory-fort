## Package Surface

This is a pre-final local package-surface smoke. It reports the current dry-run publish surface, but because Task 8 did not run `npm run build`, it does not prove that `dist/cli.mjs` or the packed CLI reflects the current TypeScript source. Final package proof is reserved for Task 10 after a fresh build.

PowerShell proof shape:

```powershell
$raw = npm pack --dry-run --json
$text = ($raw -join "`n").TrimStart([char]0xFEFF)
$data = $text | ConvertFrom-Json
$pkg = $data[0]
[pscustomobject]@{ name = $pkg.name; version = $pkg.version; fileCount = $pkg.files.Count; packageSize = $pkg.size; unpackedSize = $pkg.unpackedSize }
```
