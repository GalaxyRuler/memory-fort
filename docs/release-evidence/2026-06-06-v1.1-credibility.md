## Package Surface

This is the Task 10 final package-surface proof. It was run after a fresh `npm run build`, which completed successfully and produced `dist/hooks/opencode-event.mjs`.

PowerShell proof shape:

```powershell
$rawPackJson = npm pack --dry-run --json
$packText = ($rawPackJson -join "`n").TrimStart([char]0xFEFF)
$packData = $packText | ConvertFrom-Json
$packItem = $packData[0]
$packPaths = @($packItem.files | ForEach-Object { $_.path })
$topEntries = $packPaths | ForEach-Object {
  $first = ($_ -split '/')[0]
  if ($_ -like '*/*') { "$first/" } else { $first }
} | Sort-Object -Unique
$docsIncluded = [bool]($packPaths | Where-Object { $_ -like 'docs/*' })
$hookIncluded = [bool]($packPaths | Where-Object { $_ -eq 'dist/hooks/opencode-event.mjs' })
[pscustomobject]@{
  name = $packItem.name
  version = $packItem.version
  fileCount = @($packItem.files).Count
  packageSize = $packItem.size
  unpackedSize = $packItem.unpackedSize
  topLevelEntries = @($topEntries)
  docsIncluded = $docsIncluded
  opencodeHookIncluded = $hookIncluded
}
```

## OpenCode Temp Install Smoke

This was a focused Task 10 smoke in a disposable temp profile, not a live/operator OpenCode environment smoke.

PowerShell proof script:

```powershell
$ErrorActionPreference='Stop'
$tempBase = [System.IO.Path]::GetFullPath($env:TEMP)
$tempRoot = Join-Path $tempBase ("memory-fort-opencode-smoke-" + [guid]::NewGuid())
$oldMemoryRoot = $env:MEMORY_ROOT
$oldOpenCodeDir = $env:MEMORY_OPENCODE_DIR
$hadMemoryRoot = Test-Path Env:\MEMORY_ROOT
$hadOpenCodeDir = Test-Path Env:\MEMORY_OPENCODE_DIR
$createdRoot = $false
try {
  $resolvedTempRoot = [System.IO.Path]::GetFullPath($tempRoot)
  if (-not $resolvedTempRoot.StartsWith($tempBase, [System.StringComparison]::OrdinalIgnoreCase)) {
    throw "Temp root escaped TEMP: $resolvedTempRoot"
  }

  $env:MEMORY_ROOT = Join-Path $resolvedTempRoot ".memory"
  $env:MEMORY_OPENCODE_DIR = Join-Path $resolvedTempRoot "opencode"
  New-Item -ItemType Directory -Path $resolvedTempRoot -Force | Out-Null
  $createdRoot = $true

  node dist/cli.mjs init | Out-Null
  node dist/cli.mjs install opencode --yes --no-verify | Out-Null

  $configPath = Join-Path $env:MEMORY_OPENCODE_DIR "opencode.json"
  $pluginPath = Join-Path $env:MEMORY_OPENCODE_DIR "plugins\memory-fort.js"
  $config = Get-Content -LiteralPath $configPath -Raw | ConvertFrom-Json
  $pluginText = Get-Content -LiteralPath $pluginPath -Raw

  if ($config.mcp.memory.type -ne "local") { throw "Expected mcp.memory.type local" }
  if ($pluginText -notmatch "opencode-event\.mjs") { throw "Expected plugin hook reference" }

  [pscustomobject]@{
    TempContainedUnderTemp = $resolvedTempRoot.StartsWith($tempBase, [System.StringComparison]::OrdinalIgnoreCase)
    ConfigMcpType = $config.mcp.memory.type
    PluginReferencesOpenCodeHook = [bool]($pluginText -match "opencode-event\.mjs")
  } | ConvertTo-Json
}
finally {
  if ($hadMemoryRoot) { $env:MEMORY_ROOT = $oldMemoryRoot } else { Remove-Item Env:\MEMORY_ROOT -ErrorAction SilentlyContinue }
  if ($hadOpenCodeDir) { $env:MEMORY_OPENCODE_DIR = $oldOpenCodeDir } else { Remove-Item Env:\MEMORY_OPENCODE_DIR -ErrorAction SilentlyContinue }
  if ($createdRoot -and (Test-Path -LiteralPath $tempRoot)) {
    $resolvedForDelete = [System.IO.Path]::GetFullPath($tempRoot)
    if ($resolvedForDelete.StartsWith($tempBase, [System.StringComparison]::OrdinalIgnoreCase)) {
      Remove-Item -LiteralPath $resolvedForDelete -Recurse -Force
    }
  }
}
```
