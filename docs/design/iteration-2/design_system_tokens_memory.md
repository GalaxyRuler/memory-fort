# Design System: Memory Technical OS

## Core Colors
- **Background Base:** #09090b (Deep void)
- **Surface 1:** #111114 (Primary cards/panels)
- **Surface 2:** #18181c (Hover states/raised elements)
- **Accent:** Linear gradient (#8b5fff → #5b8bff)

## Entity Palette (Emissive)
- **Projects:** #5b8bff (Cyan-blue)
- **Decisions:** #8b5fff (Purple-violet)
- **Lessons:** #fbbf24 (Amber-gold)
- **References:** #22d3ee (Cyan-teal)
- **Tools:** #34d399 (Emerald)
- **People:** #f472b6 (Rose)
- **Crystals:** #fcd34d (Warm gold)
- **Raw-session:** #52525b (Muted gray)

## Typography
- **UI Body:** Inter (Variable), 14px base, 1.5 line-height.
- **Display:** Inter SemiBold, -0.02em letter-spacing.
- **Monospace:** JetBrains Mono, 13px (Telemetry, Code, Paths).

## Surface & Effects
- **Glass-blur:** Background: `rgba(15, 15, 20, 0.65)`, Blur: `20px`, Border: `1px solid rgba(255,255,255,0.08)`.
- **Corner Radius:** 8px (Cards), 6px (Inputs), 12px (Modals).
- **Interactive:** Suble gradient border glow on hover (20% opacity).
- **Post-processing:** UnrealBloomPass (Intensity 1.5, Threshold 0.4) for Graph nodes.
