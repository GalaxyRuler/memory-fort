# CLI

Install one platform with `memory-fort install <platform>`. Capabilities vary by platform and can include hooks, MCP entries, readiness checks, or companion files. Platform slugs tracked for v1.1 are: `claude-code` | `codex` | `antigravity` | `hermes` | `pi` | `openclaw` | `opencoven` | `opencode` | `claude-desktop` | `vscode`.

OpenCode is included as a v1.1 target and remains pending until the OpenCode implementation work lands.
